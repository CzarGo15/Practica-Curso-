Sistemas Operativos

Una breve descripción de los SO más populares.

Windows

Microsoft Windows es una familia de sistemas operativos para computadoras personales, servidores y sistemas empotrados, desarrollados y comercializados por Microsoft.

macOS

macOS es una serie de sistemas operativos gráficos desarrollados y comercializados por Apple desde 2001. Es el sistema operativo principal para la familia de computadoras Mac de Apple.

Linux

Linux es una familia de sistemas operativos de código abierto tipo Unix basados en el kernel de Linux, un núcleo de sistema operativo lanzado por primera vez el 17 de septiembre de 1991 por Linus Torvalds.
